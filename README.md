# ActionSample

## プロジェクトの説明

UE5で簡単なアクションゲームを作る

## プラットフォーム

UE5.5.4

## 操作方法

・移動　左スティック or WASD  

・カメラ　右スティック

・ジャンプ　スペース

・ドッジロール　左Ctrl

## 実装済み項目

・仮ステージ（Stage01Map）

・プレイヤ操作　ドッジロール、壁上り

・ギミック　動く足場（BP_MovingPlatform）、棒回転（BP_RollingBar）、スイングアーム（BP_SwingArm）

・プレイヤ復活

・経過時間表示

・ステージクリア演出
